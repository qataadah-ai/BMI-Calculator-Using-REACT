#BMI Calculator - React App

This is a simple BMI (Body Mass Index) Calculator built entirely by me using React.js.
It allows users to calculate their BMI by entering their weight and height.

##Features
Input Weight (kg) and Height (cm)
Click Calculate BMI button to get the result
Displays BMI value
Shows BMI category: Underweight, Normal, Overweight, Obese
Clean and simple UI

##Technologies Used

React.js (Functional components and Hooks)
CSS for styling

##Installation & Usage

Clone the repository:
git clone <your-repo-url>


Install dependencies:

npm install


Run the app:

npm start


Open in browser: http://localhost:5173
